# Placeholder directory for VRM avatars

This directory will contain VRM format anime-style avatars.

## Sources for free VRM avatars
- VRoid Hub (filter by CC0 license)
- Create your own in VRoid Studio (free software)
- OpenGameArt.org VRoid collection

## VRM Format Benefits
- Standardized expression presets (happy, sad, angry, etc.)
- Built-in blink morphs
- Smaller file sizes than unoptimized RPM
- Works offline (no API dependency)

## Typical file sizes
- 5-15MB per VRM file
